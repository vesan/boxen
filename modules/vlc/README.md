# VLC Puppet Module for Boxen
[![Build
Status](https://travis-ci.org/boxen/puppet-vlc.png?branch=master)](https://travis-ci.org/boxen/puppet-vlc)


## Usage

```puppet
include vlc
```

```puppet
class { 'vlc': 
  version => '2.1.4'
}
```

## Required Puppet Modules

None.

## Developing

Write code.

Run `script/cibuild`.
